# 🍱 Framer Style Bento Portfolio

AppleやFramerのサイトで大流行している「Bento Box（弁当箱）」レイアウトを用いた、洗練されたポートフォリオテンプレートです。

**👉 [ライブプレビューを見る (Live Demo)](https://kotao-boop.github.io/framer-template-bento/)**

## ✨ 特徴
- HTML/CSSのみで構築されており、ビルド不要で動作します。
- 漆黒の背景とグラスモーフィズムを取り入れたカードデザイン。
- マウスホバー時に滑らかに浮き上がるFramer風のアニメーション。

## 🚀 使い方
1. このリポジトリをダウンロード（ZIP）するか、フォークします。
2. `index.html` を開き、中のテキスト（名前や自己紹介）を自分のものに書き換えるだけですぐに使えます。
3. GitHub Pages等で無料で公開可能です。
